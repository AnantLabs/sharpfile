Four-D Project Page*
http://phoenix.goucher.edu/~fourd/
This was a school project that I did not originally design although I helped with some of the HTML coding (~15%). I did the bulk (~95%) of the PHP coding that drove the actual site. It links into a PostgreSQL database for the usernames and passwords and displays different content dependent on whether the user is a student, teacher, or administrator. For students, the site shows a quiz. For teachers, the site displays the scores for their respective students. The database holds the scores for all quizzes taken by the students. For administrators, the site allows them to add, modify or create users and/or questions for the quiz. For more detailed information, the proposal is here: http://phoenix.goucher.edu/~fourd/papers/proposal.html. This site requires a password to login.
